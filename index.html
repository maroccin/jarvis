⚙️ Einrichten

Alles wird nur lokal in deinem Browser gespeichert – nie online.

🌐 Live-Internet (für Trading & Geld)
🔊 Sound-Effekte (futuristisch)
🧠 Bessere Spracherkennung (Whisper, lokal, kostenlos)

Erkennt deutlich genauer als die eingebaute Browser-Erkennung, läuft komplett auf deinem Gerät (kein Konto/Kosten). Lädt beim ersten Gebrauch einmalig ein Modell (~40 MB), etwas langsamer als die Standard-Erkennung.

Noch keinen Key? Hol ihn auf console.anthropic.com → API Keys.

🏠 Philips Hue verbinden

Damit Jarvis deine echten Hue-Lampen schaltet. (Dein Laptop muss im selben WLAN wie die Hue-Bridge sein.)

📺 LG TV verbinden

Für An/Aus & Lautstärke. Der TV muss bei der ersten Kopplung eingeschaltet sein – er fragt dann „Zugriff erlauben?". (Hinweis: Einschalten aus dem Standby geht aus dem Browser leider nicht.)

🦾 Home Assistant verbinden

Verbindet ALLE deine HA-Geräte (TV, Kameras, Steckdosen) mit Jarvis – steuerbar per Klick & Sprache.

📷 Kamera per URL hinzufügen

Ohne Home Assistant: IP-Kamera, Überwachungskamera oder Snapshot-URL direkt eintragen. Die Adresse liefert dein Kamera-Hersteller/-App (meist etwas wie http://192.168.1.50/snapshot.jpg oder eine MJPEG-Stream-URL).

🎵 Spotify verbinden

Damit Jarvis Spotify per Sprache steuert: „Spiele Adele", „Pause", „Nächstes Lied". Spotify Premium erforderlich für Wiedergabe-Steuerung.

Client-ID holen: developer.spotify.com/dashboard → App erstellen → Redirect URI:

🌐 Website veröffentlichen

Kein Setup nötig! Der „🚀 Veröffentlichen"-Knopf im Website-Bereich lädt die Seite herunter und öffnet Netlify Drop – dort nur die Datei reinziehen, fertig ist der echte Link. Kein Konto, kein Token erforderlich.

Nur für Fortgeschrittene: Mit einem optionalen Token wird direkt automatisch veröffentlicht (ohne Drag&Drop) und spätere Aktualisierungen behalten dieselbe Adresse. Token holen: app.netlify.com → User settings → Applications → New access token.
